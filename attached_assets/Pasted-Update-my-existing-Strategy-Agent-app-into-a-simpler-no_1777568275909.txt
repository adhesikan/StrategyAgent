Update my existing Strategy Agent app into a simpler, novice-friendly AI trading assistant while preserving all existing backend logic, broker integrations, setup generation, history, settings, and InstaTrade functionality.

Primary goal:
Make the app feel like an easy AI trading assistant, not a clunky professional platform. Add a new beginner-friendly “Goal Mode” as the default experience, while keeping the existing prompt-based setup generator as “Advanced Trade Builder.”

Do not remove existing features. Reorganize and simplify the UX.

Current app context:
- Strategy Agent generates AI trade setups from natural language
- Supports stocks, options, debit spreads, probability grades, risk preferences, broker connection, simulated/mock data, setup history, and InstaTrade/self-directed broker order submission
- User must approve orders before anything is sent
- App already has pages for Agent, Setups, Settings, Broker, Risk Profile, Trade Preferences, Universes, Scanner, Legal, Account

New UX direction:
Strategy Agent should have two main user paths:
1. Goal Mode — default, simple, novice-friendly
