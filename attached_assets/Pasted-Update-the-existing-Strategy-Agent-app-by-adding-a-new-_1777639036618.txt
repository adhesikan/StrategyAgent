Update the existing Strategy Agent app by adding a new “Opportunity Radar” module that ranks AI-generated stock/options candidate scenarios using live broker data, market data, technical signals, sentiment inputs, options liquidity, and risk guardrails.

Important:
Do not build this as a “predictive stock picker” or “buy/sell signal” tool.
Position it as “AI-ranked candidate scenarios for user review.”
Do not implement autonomous trading.
Every live order must require explicit user review and confirmation.

Scoring engine:
Create or update a scoring service that calculates:

finalOpportunityScore =
  technicalScore * 0.30 +
  sentimentScore * 0.20 +
  momentumScore * 0.20 +
  liquidityScore * 0.15 +
  riskScore * 0.15

Map final score to grade:
- 90–100: A+
- 80–89: A
- 70–79: B
- 60–69: C
- below 60: Do not show by default
