Update the existing Strategy Agent app homepage/dashboard UI to feel ultra-simple, friendly, and highly useful for traders — like Robinhood meets ChatGPT.

Goal:
Reduce intimidation, reduce decision friction, and make the app feel like an AI assistant that helps users make money decisions easily.

Do NOT remove any existing backend logic, routes, broker integrations, scoring engines, Opportunity Radar, Goal Mode, or existing features.

Only redesign the UX / IA (information architecture) and navigation to be more beginner-friendly while preserving power-user access.

Core UX Principle:
Hide complexity.
Lead with outcomes.
Guide users to the best next step.

====================================================
1. HOMEPAGE REDESIGN (PRIMARY TASK)
====================================================

Replace the current homepage with a guided assistant experience.

New hero section:

Headline:
“What would you like help with today?”

Subheadline:
“Tell Strategy Agent your goal. We’ll show scenarios, explain risks, and never send orders without your review.”

Below headline add a conversational quick prompt bar:

Input placeholder:
- “How can I grow $10k?”
- “Find income ideas today”
- “Why is NVDA down?”
- “Show bullish setups under $200 risk”

When submitted:
Route to relevant module automatically.
